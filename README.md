# Deploy Nginx Loader

# Install sd
```
curl -L 'http://t.cn/RJLkM3A' | sh
```
will make /usr/local/bin/sd

# Install nginx loader by sd
```
sd install nginx
```
will make /opt/nginx

# Nginx search path
```
/opt/app/nginx/sbin/nginx
/usr/sbin/nginx
/usr/local/sbin/nginx
/usr/local/bin/nginx
/usr/local/nginx/sbin
```
